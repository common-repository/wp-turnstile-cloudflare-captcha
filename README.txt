Add Cloudflare Turnstile Captcha to WordPress, WooCommerce & more. The user-friendly, reCAPTCHA replacement. 100% free!

== Description ==

Really simple captcha add all your WordPress website forms using Cloudflare Turnstile to protect them from spam! Everything works on a fast and easy. Feel no delay because your time is precious!

## PLUGIN USE CASES: ##

**Enable Captcha on below wordpress forms:**

* Login Form 
* Registration Form 
* Password Reset Form 
* Comments Form 

**Enable Captcha on below WooCommerce forms:**

* Checkout
* Login Form 
* Registration Form
* Password Reset Form

= What is CLOUDFLARE TURNSTILE? = 

Cloudflare Turnstile delivers frustration-free, CAPTCHA-free web experiences to website visitors.

What's more, Turnstile stops abuse and confirms visitors are real without the data privacy concerns or awful UX that CAPTCHAs thrust on users.
